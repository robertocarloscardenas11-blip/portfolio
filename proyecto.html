App móvil e IA conversacional

Composición abstracta (móvil y nodos) que resume el flujo entre la app, las APIs y el material de referencia que alimenta las respuestas.

Vista interactiva · arrastra para girar

Proyecto académico orientado a ofrecer contención informativa mediante conversación asistida: el usuario interactúa en lenguaje natural y el sistema consulta una base de conocimiento curada por administradores.

La arquitectura separa la experiencia móvil (Flutter), las políticas de uso de la API y el procesamiento de documentos de referencia, con foco en trazabilidad básica de sesiones y mensajes para revisión.
