Proyecto académico destacado

IA de Apoyo Psicológico — aplicación móvil desarrollada en Flutter con integración de APIs de IA para conversación asistida.

Proyecto orientado a ofrecer contención informativa mediante conversación asistida: el usuario interactúa en lenguaje natural y el sistema consulta una base de conocimiento curada por administradores. La arquitectura separa la experiencia móvil (Flutter), las políticas de uso de la API y el procesamiento de documentos de referencia, con foco en trazabilidad básica de sesiones y mensajes para revisión.
